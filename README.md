## What We Built

- **Live Dashboard:** Real-time building occupancy with WebSocket-powered updates and live activity feed
- **AI Analytics:** Utilization charts, heatmaps, and 7-day forecasting powered by Anthropic Claude
- **Revenue Tracking:** Monitor rental income by building and room type with opportunity detection
- **Anomaly Detection:** AI model automatically flags ghost bookings, phantom usage, and space hoarding
- **AI Room Search:** Natural language room search, "quiet room for 6 with a projector near the library"
- **Smart Booking:** Room booking with conflict detection, availability timeline, and instant cancellation
- **Face Recognition:** Camera-based access control with real-time unauthorized access alerts to Discord

## Tech Stack

- **Frontend:** React + Vite, Tailwind CSS v4, Recharts, Axios
- **Backend:** FastAPI, SQLite, Python
- **AI:** Anthropic Claude API
- **Access Control:** face_recognition, OpenCV
- **Real-Time:** WebSockets, Discord Webhooks
